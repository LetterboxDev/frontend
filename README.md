# Letterbox Frontend

## Stack

- Ionic
- AngularJS
- Cordova

## Getting Started

`sudo npm install -g cordova`
`sudo npm install -g ionic`
`npm install`

To start:
`ionic serve`

## Build

To build iOS:
`ionic build ios`

To emulate iOS:
`ionic emulate ios`

To build iOS:
`ionic build android`

To emulate iOS:
`ionic emulate android`
